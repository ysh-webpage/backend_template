# The Code Template of Flask-based Backend

## Usage

 - Clone this
 - Install Python
 - Install `uv` by `pip` using `python3 -m pip install uv`
 - Change the ***project name*** in `pyproject.toml`
 - Run `uv update` to install dependenices

## Run

> Make sure to initialize the core subproject

```bash
uv run main.py
```
